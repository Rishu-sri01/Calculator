Java Console Calculator
=======================

Objective
---------
This project is a basic console-based calculator built using Java.
It allows the user to perform simple arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

Tools & Technologies
--------------------
- Java JDK 8 or above
- VS Code or IntelliJ Community Edition
- Terminal / Command Prompt

Project Structure
-----------------
Calculator.java   -> Main Java source code
README.txt        -> Project documentation

How to Run the Project
----------------------
1. Open terminal/command prompt in the folder where Calculator.java is saved.
2. Compile the Java program:
   javac Calculator.java
3. Run the program:
   java Calculator

Sample Output
-------------
===== Java Console Calculator =====
Enter first number: 10
Enter second number: 5
Choose an operation:
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
Enter choice (1-4): 3
Result: 10.0 * 5.0 = 50.0
Calculation complete.

Features
--------
- Simple and lightweight console application
- Handles invalid operation choice
- Prevents division by zero

Notes
-----
- The program performs one calculation per run.
- To perform another calculation, run the program again.
- This project is ideal for beginners in Java to learn about:
  * Console I/O with Scanner
  * Switch-case statements
  * Basic arithmetic operations
